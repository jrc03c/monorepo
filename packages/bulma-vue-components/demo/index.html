<bulma-button>

Default Primary Link Info Success Warning Danger White Light Dark Black Text Ghost
To apply colors, use the adjective word from the is- class name (e.g., use "danger" from the is-danger class name) as an empty attribute. For example, this: <bulma-button danger>
  Watch out!
</bulma-button> ...is equivalent to: <button class="button is-danger">
  Watch out!
</button>

<bulma-icon>

Note that the icons above have the is-large class for better visibility. Those are not the default sizes of the icons. Unlike Bulma, which recommends Font Awesome icons, this library uses Line Awesome icons. Use the name attribute to select a Line Awesome icon (without including the la- prefix in the name). For example, this: <bulma-icon class="is-large" name="question-circle"></bulma-icon> ...produces the first of the five icons shown above.
